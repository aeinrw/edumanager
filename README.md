# edumanager

基于C++ Qt和Mysql的教务管理系统.

软件工程大作业,适合GUI编程和数据库入门.

* **已经实现的模块(v1.0)**

  1. 登陆界面

  2. 管理员的管理界面(功能最为复杂的模块)
     
     * 可以一次打开4个表
     
     * 增删改查功能
  
  3. 学生和老师模块(功能较为简单,仅仅实现密码修改功能)

* **待实现的模块(可能会鸽)**
  1. 学生选课模块
  2. 老师录入成绩

数据库依赖文件(***edumanager.sql***)的导入方法

1. 进入命令行模式,输入以下命令打开mysql命令行

   `mysql -u root -p;`

2. 输入数据库密码

3. 创建数据库并使用

   `CREATE DATABASE edumanager;`
   `USE edumanager;`

4. 导入文件

   `SOURCE C:/....... ;`

   后面为edumanager.sql的绝对路径(记得分号)
